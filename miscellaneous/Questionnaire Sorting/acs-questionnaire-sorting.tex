\begin{itemize}
	\item Im Folgenden werden Sie dazu aufgefordert bestimmte Begriffe in einer vorgegebenen Reihenfolge zu sortieren.

	\item Bitte lassen Sie die kleinen Quadrate auf alle Fälle frei!
\end{itemize}
