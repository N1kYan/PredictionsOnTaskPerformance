\textbf{Aufgabe:}
\begin{itemize}
	\item Im Folgenden werden Sie dazu aufgefordert Begriffe in eine bestimmte Reihenfolge zu sortieren.
	\item Schreiben Sie dafür die Begriffe in die rechteckigen Kästchen in der Mitte jeder Aufgabe.\\Bitte lassen Sie die kleinen Quadrate auf alle Fälle frei!
	\item Nachdem Sie die Begriffe einer Aufgabe sortiert haben, zeichnen Sie bitte eine Wahrscheinlichkeitsverteilung in das große Quadrat auf der rechten Seite.
	\item Eine Wahrscheinlichkeitsverteilung ordnen einer Menge Zahlen zwischen 0 und 1 zu. In diesem Fall heißt dies, sie sollen bestimmen, wie sicher Sie sich sind, dass Sie die Aufgabe 100\% korrekt gelöst haben (=1 = 100\%sicehr, 0 = 0\% sicher). 
\end{itemize}
